php_getdns_rr_dict2str_buf

(PECL getdns >= 0.11.0)

php_getdns_rr_dict2str_buf — Convert a DNS resource record in dictionary format to string format.

Description

int php_getdns_rr_dict2str_buf ( int $dict , string &$str , int &$strLen )

This function converts a DNS resource record in dictionary format to string format. The output value is written into an existing string.

Parameters

dict

An integer representation of a getdns dictionary that contains a DNS resource record.

str

A string that will receive the resource record in string format. Note that this argument is passed by reference and any existing value will be overwritten.

strLen

On input this value should contain the length of the string. On output this value will describe the number of bytes needed for the string format representation of the resource record even if it did not fit. Note that this argument is passed by reference and any existing value will be overwritten.

Return Values

This function returns an integer response code. GETDNS_RETURN_GOOD indicates successful completion. GETDNS_RETURN_NEED_MORE_SPACE will be returned if the string was too small and strLen will be set to the number of bytes needed.
