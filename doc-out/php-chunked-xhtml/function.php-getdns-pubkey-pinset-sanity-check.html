php_getdns_pubkey_pinset_sanity_check

(PECL getdns >= 0.11.0)

php_getdns_pubkey_pinset_sanity_check — Test a given pinset for validity.

Description

int php_getdns_pubkey_pinset_sanity_check ( int $pinset , int $errorlist )

This function tests whether a given pinset is reasonable, including: Is it well-formed? Are there at least two pins? Are the digests used sane?

Parameters

pinset

An integer representation of a getdns list that contains a set of public key pins to check for sanity. This should be a list of dictionaries.

errorlist

An integer representation of a getdns list. The value may be 0 (zero) if the client does not wish to receive information about pinsets that do not pass the sanity check. A non-zero list will be modified by appending human-readable strings that describe any errors.

Return Values

This function returns an integer response code. GETDNS_RETURN_GOOD indicates that the pinset passed the sanity check.
