php_getdns_pubkey_pin_create_from_string

(PECL getdns >= 0.11.0)

php_getdns_pubkey_pin_create_from_string — Convert an HTTP Public Key Pinning (HPKP)-style pin description to a dictionary.

Description

int php_getdns_pubkey_pin_create_from_string ( int $context , string $str )

This function converts an HTTP Public Key Pinning (HPKP)-style pin description to a getdns dictionary. An example string value is: (with the quotes, without any leading or trailing whitespace):

pin-sha256="E9CZ9INDbd+2eRQozYqqbQ2yXLVKB9+xcprMF+44U1g="

Parameters

context

The previously created DNS context that is to be used with this request. The value of the context may be 0 (zero) to create a dictionary that is not associated with a particular context.

str

The pinning string to parse.

Return Values

This function returns an integer representation of a getdns dictionary. The return value will be 0 (zero) if the input string could not be converted. It is the caller's responsibility to call php_getdns_dict_destroy() on the returned dictionary when it is no longer needed.
