php_getdns_context_process_async

(PECL getdns >= 0.3.0)

php_getdns_context_process_async — Process outstanding asynchronous requests.

Description

int php_getdns_context_process_async ( int $context )

This function informs the context to process its outstanding requests. Clients should call this when either a timeout has occurred or the file descriptor signals that it is ready. User callbacks are fired during this call.

Parameters

context

The previously created DNS context that is to be used with this request.

Return Values

This function returns an integer response code. GETDNS_RETURN_GOOD indicates successful completion.
