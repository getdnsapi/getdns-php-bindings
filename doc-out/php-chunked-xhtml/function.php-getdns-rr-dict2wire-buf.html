php_getdns_rr_dict2wire_buf

(PECL getdns >= 0.11.0)

php_getdns_rr_dict2wire_buf — Convert a DNS resource record in dictionary format to binary wire format.

Description

int php_getdns_rr_dict2wire_buf ( int $dict , string &$wire , int &$wireSize )

This function converts a DNS resource record in dictionary format to binary wire format. The output value is written into an existing buffer.

Parameters

dict

An integer representation of a getdns dictionary that contains a DNS resource record.

wire

A binary string that will receive the resource record in wire format. Note that this argument is passed by reference and any existing value will be overwritten.

wireSize

On input this value should contain the size of the buffer. On output this value will describe the number of bytes needed for the wire format representation of the resource record even if it did not fit. Note that this argument is passed by reference and any existing value will be overwritten.

Return Values

This function returns an integer response code. GETDNS_RETURN_GOOD indicates successful completion. GETDNS_RETURN_NEED_MORE_SPACE will be returned if the buffer was too small and wireSize will be set to the number of bytes needed.
