php_getdns_validate_dnssec

(PECL getdns >= 0.2.0)

php_getdns_validate_dnssec — Perform DNSSEC validation without using extensions.

Description

int php_getdns_validate_dnssec ( int $toValidate , int $supportRecords , int $trustAnchors )

This function can be used by an application to have the getdns library perform DNSSEC validation without using extensions as part of a query. The getdns library will use the resource records in supportRecords to construct the validation chain and the DNSKEY or DS records in trustAnchors as trust anchors. The default list of trust anchor records that is used by the library to validate DNSSEC can be retrieved using the php_getdns_root_trust_anchor() helper function.

Parameters

toValidate

A reference to the list that represents the resource record to be validated.

supportRecords

A reference to the list that represents the records used to construct the validation chain.

trustAnchors

A reference to the list that represents the trust anchor records to use for validation.

Return Values

This function returns an integer response code. GETDNS_DNSSEC_SECURE indicates successful completion.
