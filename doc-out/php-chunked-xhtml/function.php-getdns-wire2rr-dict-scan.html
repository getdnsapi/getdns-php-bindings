php_getdns_wire2rr_dict_scan

(PECL getdns >= 0.11.0)

php_getdns_wire2rr_dict_scan — Convert a DNS resource record in binary wire format to dictionary format.

Description

int php_getdns_wire2rr_dict_scan ( string &$wire , int &$wireSize , int &$dict )

This function converts a DNS resource record in binary wire format to dictionary format.

Parameters

wire

A binary string that contains the resource record in wire format. On output the buffer pointer will have moved along the buffer and point right after the first resource record. Note that this argument is passed by reference and any existing value will be overwritten.

wireSize

On input this value should contain the size of the buffer. On output this value will describe the number of bytes that remain in the buffer after subtracting the wire format representation of the resource record. Note that this argument is passed by reference and any existing value will be overwritten.

dict

An integer representation of a getdns dictionary that contains a DNS resource record. Note that this argument is passed by reference and any existing value will be overwritten. The client is responsible for calling php_getdns_dict_destroy() when the dictionary is no longer needed.

Return Values

This function returns an integer response code. GETDNS_RETURN_GOOD indicates successful completion.
