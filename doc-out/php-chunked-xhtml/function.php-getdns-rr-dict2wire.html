php_getdns_rr_dict2wire

(PECL getdns >= 0.11.0)

php_getdns_rr_dict2wire — Convert a DNS resource record in dictionary format to binary wire format.

Description

int php_getdns_rr_dict2wire ( int $dict , string &$wire , int &$wireSize )

This function converts a DNS resource record in dictionary format to binary wire format.

Parameters

dict

An integer representation of a getdns dictionary that contains a DNS resource record.

wire

A newly allocated binary string that contains the resource record in wire format. Note that this argument is passed by reference and any existing value will be overwritten.

wireSize

The size of the allocated buffer in bytes. Note that this argument is passed by reference and any existing value will be overwritten.

Return Values

This function returns an integer response code. GETDNS_RETURN_GOOD indicates successful completion.
