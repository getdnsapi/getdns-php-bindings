php_getdns_rr_dict2str_scan

(PECL getdns >= 0.11.0)

php_getdns_rr_dict2str_scan — Convert a DNS resource record in dictionary format to string format.

Description

int php_getdns_rr_dict2str_scan ( int $dict , string &$str , int &$strLen )

This function converts a DNS resource record in dictionary format to string format. The output value is written into an existing string.

Parameters

dict

An integer representation of a getdns dictionary that contains a DNS resource record.

str

A string that will receive the resource record in string format. On output the buffer pointer will have moved along the string and point right after the just written resource record. Note that this argument is passed by reference and any existing value will be overwritten.

strLen

On input this value should contain the length of the string. On output this value will describe the number of bytes that remain in the string after writing the string representation of the resource record. Note that this argument is passed by reference and any existing value will be overwritten.

Return Values

This function returns an integer response code. GETDNS_RETURN_GOOD indicates successful completion. GETDNS_RETURN_NEED_MORE_SPACE will be returned if the string was too small. The function will pretend that it had written beyond the end of the string, str will point past the buffer, and strLen will contain a negative value.
