php_getdns_fp2rr_list

(PECL getdns >= 0.11.0)

php_getdns_fp2rr_list — Convert the contents of a DNS zone file to getdns list format.

Description

int php_getdns_fp2rr_list ( string $inFile , int &$list , string $origin , int $defaultTTL )

This function converts the contents of a DNS zone file to getdns list format. This list will contain dictionaries representing the resource records found in the zone file.

Parameters

inFile

A string containing the path to the file to be opened for reading.

list

On output this value will contain an integer representation of a list of dictionaries that describe the resource records found in the zone file. Note that this argument is passed by reference and any existing value will be overwritten. The client is responsible for calling php_getdns_list_destroy() when the list is no longer needed.

origin

A string that contains the default suffix to be used for any domain names that are not fully qualified.

defaultTTL

An integer that contains the default "time to live" (TTL) value to be used in building the dictionaries.

Return Values

This function returns an integer response code. GETDNS_RETURN_GOOD indicates successful completion.
