php_getdns_dict_set_int

(PECL getdns >= 0.11.0)

php_getdns_dict_set_int — Set an integer value in a dictionary.

Description

int php_getdns_dict_set_int ( int $dict , string $name , int $value )

This function sets an integer value in a dictionary. The dictionary type is used to manage name/value pairs in which the names are strings and the data types of the values are heterogeneous and include bindata, dictionaries, lists, and integers.

Parameters

dict

The previously created dictionary that is to be used with this request.

name

The name whose value is to be set.

value

An unsigned 32-bit integer value to be associated with the given name.

Return Values

This function returns an integer response code. GETDNS_RETURN_GOOD indicates successful completion.
