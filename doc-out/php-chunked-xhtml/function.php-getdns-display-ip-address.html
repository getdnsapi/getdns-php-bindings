php_getdns_display_ip_address

(PECL getdns >= 0.1.0)

php_getdns_display_ip_address — Convert an IP address in bindata format to a character string.

Description

string php_getdns_display_ip_address ( int $bindata )

This function converts an IPv4 or IPv6 address in bindata format to a presentation-friendly character string.

Parameters

bindata

A reference to an IP address in bindata format.

Return Values

This function returns an IP address in character string format. Errors in the conversion process are indicated with a NULL return value.
