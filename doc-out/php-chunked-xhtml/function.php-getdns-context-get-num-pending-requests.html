php_getdns_context_get_num_pending_requests

(PECL getdns >= 0.3.0)

php_getdns_context_get_num_pending_requests — Get the number of pending resolution requests.

Description

int php_getdns_context_get_num_pending_requests ( int $context )

This function gets the number of pending resolution requests from a context.

Parameters

context

The previously created DNS context that is to be used with this request.

Return Values

This function returns an integer representing the number of pending requests.
