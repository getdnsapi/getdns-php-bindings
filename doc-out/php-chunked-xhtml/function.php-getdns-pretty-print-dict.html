php_getdns_pretty_print_dict

(PECL getdns >= 0.1.0)

php_getdns_pretty_print_dict — Creates a formatted character string representation of a dictionary.

Description

string php_getdns_pretty_print_dict ( int $dict )

This function can be used to create a "pretty printed" representation of a getdns dictionary.

Parameters

$dict

The dictionary to be pretty printed.

Return Values

This function returns a character string containing the pretty printed dictionary.
