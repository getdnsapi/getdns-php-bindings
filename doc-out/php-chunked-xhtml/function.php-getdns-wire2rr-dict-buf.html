php_getdns_wire2rr_dict_buf

(PECL getdns >= 0.11.0)

php_getdns_wire2rr_dict_buf — Convert a DNS resource record in binary wire format to dictionary format.

Description

int php_getdns_wire2rr_dict_buf ( string $wire , int &$wireSize , int &$dict )

This function converts a DNS resource record in binary wire format to dictionary format.

Parameters

wire

A binary string that contains a resource record in wire format.

wireSize

On input this value should contain the size of the buffer. On output this value will describe the length of the wire format resource record. Note that this argument is passed by reference and any existing value will be overwritten.

dict

On output this value will contain an integer representation of a getdns dictionary that contains a DNS resource record. Note that this argument is passed by reference and any existing value will be overwritten. The client is responsible for calling php_getdns_dict_destroy() when the dictionary is no longer needed.

Return Values

This function returns an integer response code. GETDNS_RETURN_GOOD indicates successful completion.
