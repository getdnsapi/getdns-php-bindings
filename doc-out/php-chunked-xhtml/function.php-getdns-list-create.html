php_getdns_list_create

(PECL getdns >= 0.4.0)

php_getdns_list_create — Create an empty list.

Description

int php_getdns_list_create ( void )

This function creates an empty list.

Parameters

This function has no parameters.

Return Values

This function returns an integer representation of an empty list. The list must eventually be freed using php_getdns_list_destroy(). The return value will be 0 (zero) if an error occurred.
