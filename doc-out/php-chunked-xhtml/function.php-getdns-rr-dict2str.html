php_getdns_rr_dict2str

(PECL getdns >= 0.11.0)

php_getdns_rr_dict2str — Convert a DNS resource record in dictionary format to string format.

Description

int php_getdns_rr_dict2str ( int $dict , string &$str )

This function converts a DNS resource record in dictionary format to string format.

Parameters

dict

An integer representation of a getdns dictionary that contains a DNS resource record.

str

A string representation of the DNS resource record. Note that this argument is passed by reference and any existing value will be overwritten.

Return Values

This function returns an integer response code. GETDNS_RETURN_GOOD indicates successful completion.
