php_getdns_strerror

(PECL getdns >= 0.1.0)

php_getdns_strerror — Get a character string that describes a getdns error code.

Description

int php_getdns_strerror ( int $errorValue , string &$outputString , int &$outputStringLen )

This function translates a getdns error code into a human-readable character string.

Parameters

errorValue

The getdns error code to be translated.

outputString

The character string representation of the getdns error code. Note that this argument is passed by reference and any existing value will be overwritten.

outputStringLen

The length of the character string. Note that this argument is passed by reference and any existing value will be overwritten.

Return Values

This function returns an integer response code. GETDNS_RETURN_GOOD indicates successful completion.
