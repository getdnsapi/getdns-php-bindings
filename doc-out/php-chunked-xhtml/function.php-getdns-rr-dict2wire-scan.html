php_getdns_rr_dict2wire_scan

(PECL getdns >= 0.11.0)

php_getdns_rr_dict2wire_scan — Convert a DNS resource record in dictionary format to binary wire format.

Description

int php_getdns_rr_dict2wire_scan ( int $dict , string &$wire , int &$wireSize )

This function converts a DNS resource record in dictionary format to binary wire format. The output value is written into an existing buffer.

Parameters

dict

An integer representation of a getdns dictionary that contains a DNS resource record.

wire

A binary string that will receive the resource record in wire format. On output the buffer pointer will have moved along the buffer and point right after the just written resource record. Note that this argument is passed by reference and any existing value will be overwritten.

wireSize

On input this value should contain the size of the buffer. On output this value will describe the number of bytes that remain in the buffer after writing the wire format representation of the resource record. Note that this argument is passed by reference and any existing value will be overwritten.

Return Values

This function returns an integer response code. GETDNS_RETURN_GOOD indicates successful completion. GETDNS_RETURN_NEED_MORE_SPACE will be returned if the buffer was too small. The function will pretend that it had written beyond the end of the buffer, wire will point past the buffer, and wireSize will contain a negative value.
