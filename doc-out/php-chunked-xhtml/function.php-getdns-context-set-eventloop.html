php_getdns_context_set_eventloop

(PECL getdns >= 0.1.0)

php_getdns_context_set_eventloop — Set an event loop for asynchronous processing.

Description

ReturnType php_getdns_context_set_eventloop ( int $context )

This function sets an event loop for asynchronous processing. Any current event loop is detached before setting the new event loop value.

Note that this function is not fully implemented. It currently does little more than echo string values to document processing steps.

Parameters

context

The previously created DNS context that is to be used with this request.

Return Values

This function returns an integer response code. GETDNS_RETURN_GOOD indicates successful completion.
