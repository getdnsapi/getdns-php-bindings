php_getdns_context_set_dns_transport

(PECL getdns >= 0.1.0)

php_getdns_context_set_dns_transport — Set the transport protocol to be used for DNS lookups.

Description

int php_getdns_context_set_dns_transport ( int $context , int $value )

This function sets the transport protocol to be used for for DNS lookups.

Parameters

context

The previously created DNS context that is to be used with this request.

value

The value of the transport protocol option to be set. Valid options include GETDNS_TRANSPORT_UDP_FIRST_AND_FALL_BACK_TO_TCP, GETDNS_TRANSPORT_UDP_ONLY, GETDNS_TRANSPORT_TCP_ONLY, or GETDNS_TRANSPORT_TCP_ONLY_KEEP_CONNECTIONS_OPEN.

Return Values

This function returns an integer response code. GETDNS_RETURN_GOOD indicates successful completion.
