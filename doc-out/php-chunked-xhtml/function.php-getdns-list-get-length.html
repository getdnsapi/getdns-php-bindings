php_getdns_list_get_length

(PECL getdns >= 0.7.0)

php_getdns_list_get_length — Get a count of the number of elements in a list.

Description

int php_getdns_list_get_length ( int $list , int &$length )

This function gets a count of the number of elements in a list. The list type is used to manage heterogeneous lists of objects consisting of bindata, dictionaries, lists, and integers.

Parameters

list

The previously created list that is to be used with this request.

length

The number of elements contained in the list. Note that this argument is passed by reference and any existing value will be overwritten.

Return Values

This function returns an integer response code. GETDNS_RETURN_GOOD indicates successful completion.
