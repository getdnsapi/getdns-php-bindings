php_getdns_context_run

(PECL getdns >= 0.10.0)

php_getdns_context_run — Run a context's asynchronous event loop.

Description

NULL php_getdns_context_run ( int $context )

This function blocks and runs a context's asynchronous event loop until all outstanding requests have been processed.

Parameters

context

The previously created DNS context that is to be used with this request.

Return Values

This function does not return any values.
