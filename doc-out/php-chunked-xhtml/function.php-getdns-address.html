php_getdns_address

(PECL getdns >= 0.1.0)

php_getdns_address — Asynchronously retrieve IP addresses given a host name.

Description

int php_getdns_address ( int $context , string $name , int $extensions , array $userArg , double &$transID )

This function provides an entry point into the getdns API library to asynchronously retrieve IP addresses given a host name. It always returns both IPv4 and IPv6 addresses. This function will use all of the namespaces from the context. See php_getdns_address_sync() for a description of the response data structure.

Parameters

context

The previously created DNS context that is to be used with this request.

name

The host name to resolve to an address (note that an IP address is considered invalid).

extensions

The extensions dictionary for this request; NULL or 0 (zero) if no extensions.

userArg

An indexed array that contains information used to control callback processing. The first element of the array must contain a character string representing the name of the PHP client function that will receive and process the result of the DNS query. Additional array elements are optional and may be included to be passed to the callback function. The callback function will receive the complete array for local processing with all elements included as specified.

transID

A value that is filled in by the function to identify the callback being made. This value is passed by reference and any existing value will be overwritten. It can be 0 (zero), in which case it is ignored and no value is assigned. The php_getdns_cancel_callback() function uses the transID to determine which callback is to be cancelled. If the function fails, transID is set to 0.

Return Values

This function returns an integer response code. GETDNS_RETURN_GOOD indicates successful completion.
